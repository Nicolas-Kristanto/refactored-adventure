# 🎮 Refactored-Adventure

**Refactored-Adventure** adalah sebuah repositori yang berisi koleksi mini-games interaktif yang dikembangkan menggunakan bahasa pemrograman **Python**. Proyek ini berfokus pada penerapan logika pemrograman yang bersih, manajemen *state* game, dan struktur kode yang modular.

## 🚀 Fitur Utama
- **Game Logic:** Implementasi logika permainan yang responsif.
- **Modular Code:** Kode yang telah di-*refactor* agar mudah dipelajari dan dikembangkan.
- **Interactive UI:** Penggunaan pustaka (seperti Pygame atau pustaka standar lainnya) untuk interaksi pengguna.
